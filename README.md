# ToDoLists
An android ToDo list app that uses a local SQLite database to keep track of all lists
## Features
* You can choose a representative color for your list
  * Red
  * Cyan
  * Green
  * Purple
  * Orange
  * Yellow
* You can set different states for every item in a ToDo List
  * None
  * ToDo
  * In Progress
  * Done
